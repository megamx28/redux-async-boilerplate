### Install

    $ git clone https://github.com/shanedasilva/redux-async-boilerplate.git
    $ npm install
    $ npm run dev

Open [http://localhost:3000](http://localhost:3000) in your browser to display the app.

### NPM Commands
* `npm run dev`
* `npm run lint`
* `npm run lint:fix`
* `npm run test`
* `npm run test:watch`
* `npm run clean`
* `npm run compile`
* `npm run deploy`
